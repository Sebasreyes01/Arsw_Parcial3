Conversor de Temperatura

Este conversor permite pasar una temperatura en grados centigrados a grados fahrenheit y vice versa.

Conexion por URL

Otra forma de usar el servicio es por medio de la url.
Usted puede ingresar a https://arsw-parcial3.herokuapp.com/conversor/celsius/valor para convertir de centigrados a fahrenheit, reemplazando la palabra 'valor' por el valor de la temperatura. E.g. https://arsw-parcial3.herokuapp.com/conversor/celsius/10.
De igual forma puede ingresar a https://arsw-parcial3.herokuapp.com/conversor/fahrenheit/valor para convertir de fahrenheit a centigrados, reemplazando la palabra 'valor' por el valor de la temperatura. E.g. https://arsw-parcial3.herokuapp.com/conversor/fahrenheit/50.

Conexion por JS

Para conectarse por medio de JS puede usar el cliente basado en promesas axios de la siguiente forma:

axios.get('https://arsw-parcial3.herokuapp.com//conversor/grados/valor')
.then(function (response) {
// handle success
console.log(response);
})
.catch(function (error) {
// handle error
console.log(error);
})

Reemplazando valor por el valor de la temperatura y reemplazando grados por 'celsius' o 'fahrenheit' segun sea el caso.
